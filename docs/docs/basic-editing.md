# Basic Editing

1.  Simply click edit on the document you want to make a change to.

1.  Make your change in the web editor. Here you can preview your
    changes too.

1.  Type a commit summary message for the reviewer and click the
    "Propose file change" button.

1.  This next screen shows you a "diff" (comparison) of the changes
    you are proposing. You can click the "New Pull Request"
    button here.

1.  On this final screen, you click "Create Pull Request" button to
    complete the change proposal.

1.  Your changes are now in the Pull Request queue. The maintainer of
    the project will be automatically notified of your change. Once
    your changes are reviewed and approved the changes will be
    deployed to the live site and you shall be notified of this.
