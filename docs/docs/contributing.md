# Contributing

First time contributors you will need to
[email](mailto:ashley@example.com) and request access to the docs
project.  Once access is granted you can proceed.

There are two ways to submits edits to the documentation:

-   One can [edit and make changes via the website
    GUI](../basic-editing/). This method works OK for simple small
    changes. For larger or more advanced changes the next method is
    strongly recommended.

-   [Checking out the documentation code locally](../advanced-editing/),
    editing it and submitting your changes - like any coding project.
    This way is **strongly recommended** and worth the initial effort
    required.

Also ensure to read through and reference the [writing
documentation](../writing/sheet/) article which should make things a lot
easier for new comers.
